# Roadmap

## Near Term

- more attack packs for RAG poisoning, tool-call forgery, and canary replay
- richer response detectors with structural parsing instead of keyword-only heuristics
- report comparison mode for regression tracking across commits
- Docker and compose setup for one-command local labs

## Mid Term

- provider adapters for more direct client presets
- UI drill-down for request/response diffs and hit explanations
- exportable markdown and SARIF-style reports
- multi-upstream routing policies and environment profiles

## Long Term

- plugin system for probes and detectors
- distributed benchmark execution
- replay and corpus minimization for discovered failures
