# Security Policy

AIGete is designed for authorized defensive research only.

## Intended Use

- Local lab validation
- Authorized red-team style testing of your own AI systems
- Regression testing against prompt leakage and instruction hierarchy failures

## Out of Scope

- Unauthorized access to third-party systems
- Covert exfiltration tooling
- Evasion-focused payload delivery

## Reporting

If you discover a bug in AIGete itself, open a GitHub issue with reproduction details and clearly mark whether the issue affects confidentiality, integrity, or availability.
