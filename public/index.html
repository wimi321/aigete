60-second setup

Three steps. No maze.

Start the mock upstream, start AIGete, then point Codex, Claude Code, or OpenCode at the gateway.

1. Start the lab

Run the mock upstream and the gateway locally.

npm run mock
npm start

2. Leave defaults alone

The default upstream already points at the local mock server, so new users can test without extra setup.

Use the OpenAI-style URL for Codex and OpenCode, or the Anthropic-style URL for Claude Code.
